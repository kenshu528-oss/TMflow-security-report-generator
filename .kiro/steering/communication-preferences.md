# 溝通偏好設定

## 語言偏好

### 主要溝通語言
- **優先使用繁體中文**進行所有對談和回應
- 技術術語可適當使用英文，但說明要用繁體中文
- 錯誤訊息和日誌記錄使用繁體中文
- 使用者介面文字使用繁體中文

### 文檔語言規範
- 所有使用者文檔使用繁體中文撰寫
- 程式碼註解使用繁體中文
- 變數和函數名稱使用英文（遵循 Python 慣例）
- Git commit 訊息使用繁體中文

### 技術溝通原則
- 解釋技術概念時使用繁體中文
- 提供程式碼範例時加上繁體中文註解
- 錯誤排除指導使用繁體中文
- 使用者操作說明使用繁體中文

## 回應風格

### 專業但親切
- 使用專業的技術語言
- 保持友善和耐心的語調
- 提供清楚的步驟說明
- 避免過於複雜的技術術語

### 結構化回應
- 使用清楚的標題和分段
- 重要資訊使用粗體或清單格式
- 提供具體的操作步驟
- 包含相關的程式碼範例
