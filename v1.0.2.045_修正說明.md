### 問題描述
在報告生成過程中，按鈕顯示的文字是中文「生成中...」，與整體介面的英文按鈕不一致。

### 修正方案

### 修改位置
**檔案**: `ui_modular.py`
**方法**: `generate_reports()`
**行號**: 932

**修改前**:
```python

```

**修改後**:
```python

```
