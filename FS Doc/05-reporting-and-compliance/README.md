# 05 - Reporting and Compliance

This section covers generating reports, maintaining compliance, and using reporting tools.

## Available Scripts

- **[fs-report](./fs-report/)** - Reporting system that generates HTML, CSV, and XLSX reports with customizable recipes for executive summaries, component vulnerability analysis, findings by project, and scan analysis
- **[fs-reporter](./fs-reporter/)** - PDF report generator for creating detailed vulnerability and risk reports
- **[report_license](./report_license/)** - Generates CSV reports of component licenses for project versions
